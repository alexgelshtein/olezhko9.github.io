Правила сдачи лабораторных работ

Всего за семестр нужно сдать пять лабораторных работ. При сдаче лабораторной работы после дедлайна за каждый пропуск будет отниматься по три балла за каждый долг. В конце семестра нужно сдать отчеты по каждой лабораторной работе. Централизованно грузите все отчеты в облако. В отчеты можно вставлять код и скриншоты. Для зачета или минимальной оценки нужно сдать все лабораторные работы с отчетами.

Первые три лабораторные работы нужно загружать в github pages. Остальные лабораторные работы - либо на удаленном сервере, либо локально.

При сдаче лабораторных работ будет учитываться история коммитов (могу посмотреть диффы). И лучше написать свой кривой код, чем копипастить чужое. При обнаружении копипасты могу поставить ноль баллов за лабораторную работу без возможности исправления. По любой строчке в коде может быть задан вопрос, не пишите и не копируйте, чего не понимаете.
